JavaScript (User's List) exercise

The Required Functionality

  • The action buttons provides the following functionality:
    • Adding a new user
    • Editing an existing selected user (enabled only if one row is selected. Otherwise button should be disabled)
    • Deleting all selected users (if no user is selected - button should be disabled)
  • The check-box on the header row - selects all rows
  • The application validates the user's input is correct (all fields are required, age must be a number)
  • The list of users is saved in the local browser

Implementation details

  • You may use the internet as a resource, but we trust you to do this exercise without outside help!
  • You are welcome to use any popular familiar JS frameworks to implement the functionality of this page.
  • Have the user be represented by a Model / ViewModel / WhatEver (the specific framework semantics is less relevant)
  • Make sure your code is organized, fully documented & keep it simple as much as possible.
  • Persist the users data on the local user machine using cookies. (so when the page is refreshed the last list of users I created will be displayed)
  • We used here Bootstrap and JQuery to structure the page, but you may modify the html as much as you need - the functionality should stay the same
