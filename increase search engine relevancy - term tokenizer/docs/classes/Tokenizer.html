tokenize_single_words()

tokenize_single_words(string  $input_text)

tokenize single words

this function takes known words out of input text function parameter and saves them into result arrays and then returns the a result. this is a simplified function that uses strtok and in_array to match for words found in arrays in settings property.

example return value from this function:


array(
    "status" => "success",
    "original" => "red dress",
    "keywords" => "dress",
    "properties" =>
    array(
        "color"  => array( "red" ),
        "gender" => array(),
        "pattern"=> array(),
        "brand"  => array()
    )
);

example:


   include "tokenizer.class.php";
   $filename="tokenizer-gap.json";
   $tokenizer=new Tokenizer($filename);

   $query="red dress"
   $result=$tokenizer->tokenize_single_words($query);

   echo $result["keywords"]."\n\n";
   var_dump($result["parameters"]["colors"]);

Parameters

string $input_text

several words separated by spaces to match for each of them.

tokenize()

tokenize(string  $input_text)

tokenize

this function takes known words out of input text function parameter and saves them into result arrays and then returns the a result. this is a more complex function that relies on regular expression searches to match for words found in arrays in settings property.

example return value from this function:


array(
    "status" => "success",
    "original" => "red dress",
    "keywords" => "dress",
    "properties" =>
    array(
        "color"  => array( "red" ),

        "pattern"=> array(),
        "brand"  => array()

   include "tokenizer.class.php";
